info 三步推断法说明

推断原理：通过对比"有上下文"和"无上下文"两种情况下对词条含义的推断结果来判断是否为黑话。如果两个推断结果相似，说明词条含义明确（不是黑话）；如果不同，说明需要上下文才能理解（是黑话）。

渐进式验证：在词条出现次数达到 3, 6, 10, 20, 40, 60, 100 次时分别进行推断验证，随着上下文增多，推断结果更加准确。
